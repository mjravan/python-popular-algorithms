# a1z26 cipher Algorithm
A cipher algorithm for encrypting message or data.<br>
````
#a1z26 cipher
    #encoding:
    print(encode('majed')) => [109, 97, 106, 101, 100]
    #decoding:
    decode([109, 97, 106, 101, 100]) => majed
````
