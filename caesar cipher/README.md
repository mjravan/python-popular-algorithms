# Caesar Cipher Algorithm
An interesting cipher algorithm for encrypting password or texts. 
```
Caesar Cipher
    encrypt('amir', 4) => eqmv
    decrypt('eqmv', 4) => amir
```
---
# Big O Notation Of Algorithm
The algorithm’s run–time complexity is O(N)
